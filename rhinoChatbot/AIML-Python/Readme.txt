Main program: rhinoChatbot.py

Have all these files in same folder, change the path accordingly inside the main program before running it.

std.start-up.xml - Startup file which loads all AIML files for the chatbot to learn all categories and develop a brain.

rhinoModelCreate.aiml - AIML categories to create a model.
rhinoModelMove.aiml - AIML categories to move a model.
rhinoModelRotate.aiml - AIML categories to rotate a model.
rhinoModelScale.aiml - AIML categories to scale a model.

NOTE: This AIML based chatbot is not connected to Rhino3d and therefore cannot be used for modelling. It is just an user-interface.
